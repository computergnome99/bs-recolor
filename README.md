# bs-recolor
### Version: 0.0.1a
### Author: Calvin Bonner
---

To install and make updates to this project, you will need to:

1. Run `npm install`
2. Use a SCSS compiler like "Live Sass Compiler" *(for VSCode)*
3. Run the webapp in a local server *(because of AJAX functions)*
